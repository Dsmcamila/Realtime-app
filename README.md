# Realtime-app


Es una app de chat en tiempo real. Dependencias utilizadas:
 - Vanilla JS
 - Express
 - Socket.io
 - Moment
 - Redis
# Antes de empezar
Configura el entonrno para nodejs:

## Uso

```
$ npm install
```

### Inicio:

```
$ npm run dev
```
#### DB:

```
$ npm install pg
```


Ir a localhost:3000
